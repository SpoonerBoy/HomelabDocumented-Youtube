## 🏠 What's in the Lab?

I'm Delbert Gray (SpoonerBoy), running a self-hosted homelab focused on practical, reproducible setups. From Raspberry Pi clusters to full Proxmox virtualization, networking, monitoring, and self-hosted services.

**Core Focus:**
- **Documentation as Code** — Everything version-controlled, diagrammed, and explained.
- Beginner-to-advanced progression
- Real-world troubleshooting and lessons learned

### Current Stack Highlights
- **Hardware**: Raspberry Pi cluster, servers, networking gear
- **Hypervisor**: Proxmox
- **Orchestration**: Docker / Docker Compose / Kubernetes
- **IaC**: Ansible, Terraform (coming soon)
- **Monitoring**: Prometheus, Grafana, etc.
- **Documentation Tools**: Obsidian, Git, diagrams
